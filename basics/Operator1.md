### 示例
```javascript
var num = 0;
num = num + 2;
num = num * 3;
num = num / 2;
num++;
num--;

num += 1; 
num -= 2;
num *= 3;
num /= 2;
num %= 3;

console.log('num == 1 : ' + (num == 1)); 
console.log('num === 1 : ' + (num === 1));
console.log('num != 1 : ' + (num != 1));
console.log('num > 1 : ' + (num > 1));
console.log('num < 1 : ' + (num < 1));
console.log('num >= 1 : ' + (num >= 1));
console.log('num <= 1 : ' + (num <= 1));

console.log('true && false : ' + (true && false)); 
console.log('true || false : ' + (true || false));
console.log('!true : ' + (!true));
```

### 算术运算符
> +:加  
> -：减  
> *：乘  
> /：除  
> %：取余  
> ++：递增  
> --：递减  

### 赋值运算符
> =：赋值  
> +=：加赋值，`x+=y == x=x+y`  
> -=：减赋值，`x-=y == x=x-y`  
> *=：乘赋值，`x*=y == x=x*y`  
> /=：除赋值，`x/=y == x=x/y`  
> %=：取余赋值，`x%=y == x=x%=y`  

### 比较运算符
> ==： 等于  
> ===：全等于  
> >：大于  
> >=：大于等于  
> <：小于  
> <=: 小于等于

### 逻辑运算符
> &&：与
> ||：或
> !： 非
